使用方法：

- 一次性安装

  ```python
  # 安装opencv时
  pip install https://pypi.tuna.tsinghua.edu.cn/simple opencv-python
  ```

- 设置默认镜像

  ```python
  pip config set pip config set global.index-url https://pypi.tuna.tsinghua.edu.cn/simple
  ```
